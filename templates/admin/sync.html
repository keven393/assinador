Sincronização Completa

Sincroniza todos os usuários do Active Directory que correspondem ao filtro configurado. Usuários novos serão criados INATIVOS e SEM PERMISSÕES - você precisará ativar e definir permissões manualmente em Gerenciar Usuários.

Sincronização Individual

Sincroniza um usuário específico do Active Directory.

Como Funciona
Usuários Novos
  • São criados automaticamente no banco de dados
  • São criados INATIVOS (is_active=False)
  • Recebem role padrão: user (sem permissões)
  • Precisam de ativação e designação de permissões pelo admin
  • Informações são importadas do AD
Usuários Existentes
  • Informações são atualizadas do AD
  • Status de ativação é sincronizado
  • Permissões (role) são mantidas
  • Usuários não encontrados no AD são desativados
